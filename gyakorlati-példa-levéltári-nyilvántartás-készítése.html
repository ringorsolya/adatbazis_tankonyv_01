Adatbázisépítés I. - Bevezetés az adatbázisokba

5 Gyakorlati példa: Levéltári nyilvántartás készítése

5.1 A feladat

Digitális nyilvántartás készítése

Mezők: 1. Jelzet 2. Fond megnevezése 3. Kezdő év 4. Záró év 5. Terjedelem (iratfolyóméter) 6. Raktári épület 7. Digitalizálva (Igen/Nem/Részben)

5.2 Előkészületek

5.2.1 Excel megnyitása és mentés

  1. Indítsuk el az Excelt
  2. File → Save As
  3. Név: Nyilvantartas_JELZET.xlsx
  4. Save

5.2.2 Munkalap átnevezése

  1. Dupla klikk “Sheet1” fülre (lent)
  2. Írd: Fondok
  3. Enter

5.3 Fejléc sor

5.3.1 Oszlopnevek beírása

Kattints A1-re, majd írd be (Tab-bal ugorj):

A1: Jelzet
B1: Fond_megnevezése
C1: Kezdő_év
D1: Záró_év
E1: Terjedelem_ifm
F1: Raktári_épület
G1: Digitalizálva

5.3.2 Fejléc formázása

  1. Jelöld ki A1:G1
  2. Bold (Ctrl+B)
  3. Font size: 12
  4. Fill Color: Világoskék
  5. Center (középre)
  6. Borders → Bottom Border

5.3.3 Oszlopszélesség

Minden oszlopnál: vidd az egeret az oszlopok határára (A-B, B-C…) → Dupla klikk (automatikus szélesség)

Mentés: Ctrl+S

5.4 Adatok felvitele

5.4.1 Első fond

Kattints A2-re, írd be (Tab-bal tovább):

A2: XXV-1
B2: Tanácsi ülések jegyzőkönyvei
C2: 1950
D2: 1990
E2: 5
F2: I. épület
G2: Igen

Az utolsó mezőnél Enter → ugrik A3-ra (következő fond).

5.4.2 További fondok (min. 10 db)

Mintaadatok:

Jelzet Megnevezés Kezdő Záró Terjedelem (ifm) Raktár Digitalizálva
XXVI-I-3 Magyar Állami Operaház 1945 1989 9 I. épület Igen
XXVI-I-4 Állami Hangverseny- és Műsorigazgatóság 1956 1964 5 II. épület Nem
XXVI-I-5 Magyar Állami Népi Együttes 1950 1986 6.6 I. épület Részben
XXVI-I-6 Huszonötödik Színház 1969 1977 1.9 II. épület Nem
XXVI-I-7 Állami Déryné Színház 1951 1977 4 I. épület Igen
XXVI-I-8 Nemezeti Színház 1933 1997 17.85 III. épület Nem
XXVI-I-9 Pesti Hírlap 1992 1984 0.72 II. épület Részben
XXVI-I-10 Műcsarnok 1952 1992 14.64 I. épület Nem
XXVI-I-23 Magyar Filmtudományi Intézet és Filmarchívum 1960 1977 6.2 III. épület Igen
XXVI-I-65 Színház- és Filmtudományi Intézet 1957 1959 0.6 II. épület Részben

Tipp: Legyünk következetesek! “I. épület” (pont után szóköz) vagy “I.épület”? Válasszunnk egyet és alkalmazzuk mindigy ugyanúgy!

Mentés: Ctrl+S

5.5 Formázás

5.5.1 Table formázás

  1. Kattintsunk bárhova az adattáblán
  2. Home → Format as Table
  3. Válasszunk stílust (pl. kék)
  4. ☑ “My table has headers”
  5. OK

Eredmény: - Alternatív sor színek - Szűrő nyilak (▼) - Dinamikus tábla

5.5.2 Freeze Panes

  1. Kattintsunk A2-re
  2. View → Freeze Panes → Freeze Panes
  3. Görgetéskor a fejléc látszik!

Mentés: Ctrl+S

5.6 Szűrés és rendezés

5.6.1 Rendezés időrendi sorrendbe

  1. Kattintsunk “Kezdő_év” nyílra (▼)
  2. Sort Smallest to Largest

5.6.2 Szűrés - Egy épület

  1. Kattintsunk “Raktári_épület” nyílra
  2. Pipáljuk ki (Select All)
  3. Pipáljuk be csak: I. épület
  4. OK

Csak I. épület fondok látszanak!

Törlés: Ugyanaz a nyíl → Clear Filter

5.6.3 Komplex szűrés - Nem digitalizált, régi fondok

1. feltétel: Kezdő_év < 1960 - “Kezdő_év” nyíl → Number Filters → Less Than… → 1960 → OK

2. feltétel: Digitalizálva = Nem - “Digitalizálva” nyíl → Pipáljuk be csak “Nem” → OK

Eredmény: Régi, nem digitalizált fondok (2-3 db)

Törlés: Data → Clear

5.7 Adatvalidáció

5.7.1 Validáció - Évek (1800-2024)

  1. Jelöljük ki C2:C100 (Kezdő_év)
  2. Data → Data Validation
  3. Settings:
    • Allow: Whole number
    • Data: between
    • Minimum: 1800
    • Maximum: 2024
  4. Error Alert:
    • Title: “Hibás év”
    • Message: “1800-2024 között!”
  5. OK

Ismételd meg D2:D100-ra (Záró_év)!

5.7.2 Dropdown - Raktári épület

  1. Írd le I1:I3-ba:

    I1: I. épület
I2: II. épület
I3: III. épület

  2. Jelöld ki F2:F100

  3. Data → Data Validation

  4. Allow: List

  5. Source: =$I$1:$I$3

  6. OK

Most dropdown lista van!

5.7.3 Dropdown - Digitalizálva

  1. Írjuk le J1:J3-ba:

    J1: Igen
J2: Nem
J3: Részben

  2. Jelöljük ki G2:G100

  3. Data → Data Validation → List → Source: =$J$1:$J$3 → OK

Mentés: Ctrl+S

5.8 Statisztikák

5.8.1 Alapstatisztikák

Üres cellákba (pl. A13-tól):

A13: Fondok száma:
B13: =COUNTA(A2:A11)

A14: Átlagos terjedelem:
B14: =AVERAGE(E2:E11)

A15: Legrégebbi:
B15: =MIN(C2:C11)

A16: Legfiatalabb:
B16: =MAX(C2:C11)

5.8.2 Feltételes számolás

A17: Digitalizált (Igen):
B17: =COUNTIF(G2:G11,"Igen")

A18: Részben:
B18: =COUNTIF(G2:G11,"Részben")

A19: Nem digitalizált:
B19: =COUNTIF(G2:G11,"Nem")

Mentés: Ctrl+S

5.9 Vizualizáció

5.9.1 Pivot tábla - Épületek szerinti megoszlás

  1. Kattintunk az adattáblán
  2. Insert → PivotTable
  3. New Worksheet → OK
  4. Pivot Fields panel (jobbra):
    • Húzzuk “Raktári_épület” → Rows
    • Húzzuk “Raktári_épület” → Values

Eredmény:

Raktári_épület     Count
I. épület          4
II. épület         3
III. épület        2

5.9.2 Grafikon

  1. Kattintunk a Pivot táblára
  2. Insert → Column Chart
  3. Válasszuk oszlopdiagramot

Testreszabás: - Chart Title: “Fondok megoszlása raktári épületek szerint”

5.9.3 Munkalap átnevezése

Dupla klikk “Sheet1” fülre → Írjuk be: Statisztika

Mentés: Ctrl+S (utoljára!)

5.10 Gyakori hibák

Szűrés nem működik → Üres sorok vannak
Validáció nincs → Rossz tartomány
COUNTA rossz → Fejléc benne van a tartományban
Freeze Panes nem működik → Rossz cellán állunk (A2 kell!)

Miről volt szó ebben a részben: 1. Strukturált tábla (fejléc, sorok, nincs üres) 2. Formázás (Table, Freeze Panes) 3. Szűrés és rendezés 4. Adatvalidáció (számok, dropdown) 5. Statisztikák (COUNTA, COUNTIF, AVERAGE) 6. Vizualizáció (Pivot, grafikon)
