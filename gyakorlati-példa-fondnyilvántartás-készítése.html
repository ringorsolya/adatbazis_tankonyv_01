Adatbázisépítés I. - Bevezetés az adatbázisokba

Fejezet 5 Gyakorlati példa: Fondnyilvántartás készítése

Most az eddig tanultakat gyakorlatban alkalmazzuk. Készítünk egy levéltári fond nyilvántartást Excelben, lépésről lépésre.

5.1 A feladat

Digitális nyilvántartás levéltári fondokról.

Mezők: 1. Fondjel (pl. XXV-1) 2. Fond megnevezése 3. Kezdő év 4. Záró év 5. Terjedelem (iratfolyóméter) 6. Raktári épület 7. Digitalizálva (Igen/Nem/Részben)

5.2 Előkészületek

5.2.1 1. Excel megnyitása és mentés

  1. Indítsd az Excelt
  2. File → Save As
  3. Név: Fondnyilvantartas_NEVED.xlsx
  4. Save

5.2.2 2. Munkalap átnevezése

  1. Dupla klikk “Sheet1” fülre (lent)
  2. Írd: Fondok
  3. Enter

5.3 Fejléc sor

5.3.1 3. Oszlopnevek beírása

Kattints A1-re, majd írd be (Tab-bal ugorj):

A1: Fondjel
B1: Fond_megnevezése
C1: Kezdő_év
D1: Záró_év
E1: Terjedelem_ifm
F1: Raktári_épület
G1: Digitalizálva

5.3.2 4. Fejléc formázása

  1. Jelöld ki A1:G1
  2. Bold (Ctrl+B)
  3. Font size: 12
  4. Fill Color: Világoskék
  5. Center (középre)
  6. Borders → Bottom Border

5.3.3 5. Oszlopszélesség

Minden oszlopnál: vidd az egeret az oszlopok határára (A-B, B-C…) → Dupla klikk (automatikus szélesség)

Mentés: Ctrl+S

5.4 Adatok felvitele

5.4.1 6. Első fond

Kattints A2-re, írd be (Tab-bal tovább):

A2: XXV-1
B2: Tanácsi ülések jegyzőkönyvei
C2: 1950
D2: 1990
E2: 5
F2: I. épület
G2: Igen

Az utolsó mezőnél Enter → ugrik A3-ra (következő fond).

5.4.2 7. További fondok (min. 10 db)

Mintaadatok:

Fondjel Megnevezés Kezdő Záró Terjedelem Raktár Digitalizálva
XXV-1 Tanácsi ülések 1950 1990 5 I. épület Igen
XXV-2 Polgármesteri 1960 1989 3 II. épület Nem
XXV-3 Műszaki osztály 1955 1992 8 I. épület Részben
XXV-4 Pénzügyi osztály 1958 1990 6 II. épület Nem
XXV-5 Kulturális osztály 1962 1988 4 I. épület Igen
XXV-6 Oktatási osztály 1960 1991 7 III. épület Nem
XXV-7 Egészségügyi 1955 1989 5 II. épület Részben
XXV-8 Szociális osztály 1965 1990 3 I. épület Nem
XXV-9 Építésügyi 1958 1992 9 III. épület Igen
XXV-10 Mezőgazdasági 1950 1988 6 II. épület Részben

Tipp: Következetes legyél! “I. épület” (pont után szóköz) vagy “I.épület”? Válassz egyet!

Mentés: Ctrl+S

5.5 Formázás

5.5.1 8. Table formázás

  1. Kattints bárhova az adattáblán
  2. Home → Format as Table
  3. Válassz stílust (pl. kék)
  4. ☑ “My table has headers”
  5. OK

Eredmény: - Alternatív sor színek - Szűrő nyilak (▼) - Dinamikus tábla

5.5.2 9. Freeze Panes

  1. Kattints A2-re
  2. View → Freeze Panes → Freeze Panes
  3. Görgetéskor a fejléc látszik!

Mentés: Ctrl+S

5.6 Szűrés és rendezés

5.6.1 10. Rendezés időrendi sorrendbe

  1. Klikk “Kezdő_év” nyílra (▼)
  2. Sort Smallest to Largest

Most 1950-es fondok elől!

5.6.2 11. Szűrés - Egy épület

  1. Klikk “Raktári_épület” nyílra
  2. Pipáld ki (Select All)
  3. Pipáld be csak: I. épület
  4. OK

Csak I. épület fondok látszanak!

Törlés: Ugyanaz a nyíl → Clear Filter

5.6.3 12. Komplex szűrés - Nem digitalizált, régi fondok

1. feltétel: Kezdő_év < 1960 - “Kezdő_év” nyíl → Number Filters → Less Than… → 1960 → OK

2. feltétel: Digitalizálva = Nem - “Digitalizálva” nyíl → Pipáld be csak “Nem” → OK

Eredmény: Régi, nem digitalizált fondok (2-3 db)

Törlés: Data → Clear

5.7 Adatvalidáció

5.7.1 13. Validáció - Évek (1800-2024)

  1. Jelöld ki C2:C100 (Kezdő_év)
  2. Data → Data Validation
  3. Settings:
    • Allow: Whole number
    • Data: between
    • Minimum: 1800
    • Maximum: 2024
  4. Error Alert:
    • Title: “Hibás év”
    • Message: “1800-2024 között!”
  5. OK

Ismételd meg D2:D100-ra (Záró_év)!

5.7.2 14. Dropdown - Raktári épület

  1. Írd le I1:I3-ba:

    I1: I. épület
I2: II. épület
I3: III. épület

  2. Jelöld ki F2:F100

  3. Data → Data Validation

  4. Allow: List

  5. Source: =$I$1:$I$3

  6. OK

Most dropdown lista van!

5.7.3 15. Dropdown - Digitalizálva

  1. Írd le J1:J3-ba:

    J1: Igen
J2: Nem
J3: Részben

  2. Jelöld ki G2:G100

  3. Data → Data Validation → List → Source: =$J$1:$J$3 → OK

Mentés: Ctrl+S

5.8 Statisztikák

5.8.1 16. Alapstatisztikák

Üres cellákba (pl. A13-tól):

A13: Fondok száma:
B13: =COUNTA(A2:A11)

A14: Átlagos terjedelem:
B14: =AVERAGE(E2:E11)

A15: Legrégebbi:
B15: =MIN(C2:C11)

A16: Legfiatalabb:
B16: =MAX(C2:C11)

5.8.2 17. Feltételes számolás

A17: Digitalizált (Igen):
B17: =COUNTIF(G2:G11,"Igen")

A18: Részben:
B18: =COUNTIF(G2:G11,"Részben")

A19: Nem digitalizált:
B19: =COUNTIF(G2:G11,"Nem")

Mentés: Ctrl+S

5.9 Vizualizáció

5.9.1 18. Pivot tábla - Épületek szerinti megoszlás

  1. Kattints az adattáblán
  2. Insert → PivotTable
  3. New Worksheet → OK
  4. Pivot Fields panel (jobbra):
    • Húzd “Raktári_épület” → Rows
    • Húzd “Raktári_épület” → Values

Eredmény:

Raktári_épület     Count
I. épület          4
II. épület         3
III. épület        2

5.9.2 19. Grafikon

  1. Kattints a Pivot táblára
  2. Insert → Column Chart
  3. Válassz oszlopdiagramot

Testreszabás: - Chart Title: “Fondok megoszlása raktári épületek szerint”

5.9.3 20. Munkalap átnevezése

Dupla klikk “Sheet1” fülre → Írd: Statisztika

Mentés: Ctrl+S (utoljára!)

5.10 Összefoglalás

Gratulálok! Elkészítetted az első adatbázis-szerű Excel táblát!

Mit tanultunk: 1. Strukturált tábla (fejléc, sorok, nincs üres) 2. Formázás (Table, Freeze Panes) 3. Szűrés és rendezés 4. Adatvalidáció (számok, dropdown) 5. Statisztikák (COUNTA, COUNTIF, AVERAGE) 6. Vizualizáció (Pivot, grafikon)

Mi a következő lépés?

Következő órán: Accessbe importálás, relációs adatbázis építés!

Házi feladat: 1. Adj hozzá még 5 fondot 2. Próbálj ki különböző szűréseket 3. Készíts egy másik Pivot táblát: “Digitalizálás megoszlása”

5.11 Gyakori hibák

⚠️ Szűrés nem működik → Üres sorok vannak
⚠️ Validáció nincs → Rossz tartomány
⚠️ COUNTA rossz → Fejléc benne van a tartományban
⚠️ Freeze Panes nem működik → Rossz cellán állsz (A2 kell!)
